# GAM531 Midterm Challenge : “Mini 3D Explorer Game”

## Description
Explore a textured room with 3 illuminated cubes. Toggle lights with mouse buttons!

## Gameplay Instructions
- **WASD**: Move forward/back/left/right
- **QE**: Move up/down
- **Mouse**: Look around (drag)
- **Mouse Wheel**: Zoom FOV
- **LMB**: Toggle Light 1 (box)
- **RMB**: Toggle Light 2 (crystal)
- **MMB**: Toggle Light 3 (gold)
- **Esc**: Quit

## Features
- 3 cube meshs in a large room
- **Phong lighting**: 3 toggleable point lights (toggle with mouse button: left click, right click and mouse wheel click).
- 5 **textures** (royalty-free) includes 3 cube texture ( crate, gold, crystal ), plain grey-ish walls and concrete floor.
- Camera: Mouse/keyboard/FOV/bounds

## Build & Run
1. Clone repo
2. Open `Game.sln` in Visual Studio 2022
3. **NuGet** (should already be pre-installed with repository clone, but if not): install OpenTK, StbImageSharp
4. Build → Run
